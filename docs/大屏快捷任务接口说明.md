# 大屏快捷任务接口说明

文档状态：接口草案  
编制日期：2026-07-23

关联文档：

- [大屏快捷任务需求说明](大屏快捷任务需求说明.md)

## 1. 说明

本接口由大屏 BFF 提供给前端。BFF 负责调用管理端任务接口，前端不直接依赖管理端内部地址。

实际传输使用标准 JSON，示例中的注释仅用于说明。

## 2. 查询机器人快捷任务

```text
GET /api/bigscreen/panorama/devices/{robotId}/quick-tasks
```

响应：

```jsonc
{
  "robotId": "robot-001",
  "currentMapId": "map-floor-2",
  "items": [
    {
      "taskPlanId": 2102,
      "taskCode": "RETURN_HOME",
      "displayName": "返回二楼驻点",
      "taskType": "NAVIGATION",
      "available": true,
      "unavailableReasonCode": null,
      "unavailableReason": null,
      "requiresConfirm": true,
      "confirmText": "确认让机器人返回二楼驻点？"
    }
  ],
  "timestamp": "2026-07-23T10:30:00+08:00"
}
```

字段：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| `robotId` | string | 是 | 机器人 ID |
| `currentMapId` | string/null | 否 | 当前地图 |
| `items` | array | 是 | 快捷任务列表 |
| `items[].taskPlanId` | number | 是 | 管理端任务计划 ID |
| `items[].taskCode` | string | 是 | `RETURN_HOME/DOCK/UNDOCK` |
| `items[].displayName` | string | 是 | 页面按钮名称 |
| `items[].taskType` | string | 是 | 任务分类 |
| `items[].available` | boolean | 是 | 当前是否可执行 |
| `items[].unavailableReasonCode` | string/null | 是 | 不可执行原因编码 |
| `items[].unavailableReason` | string/null | 是 | 页面提示 |
| `items[].requiresConfirm` | boolean | 是 | 是否二次确认 |
| `items[].confirmText` | string/null | 否 | 确认文案 |

未绑定的任务不返回；已绑定但暂时不能执行的任务返回 `available=false`。

## 3. 创建任务执行实例

```text
POST /api/bigscreen/panorama/devices/{robotId}/task-executions
```

请求：

```jsonc
{
  "taskPlanId": 2102,
  "idempotencyKey": "web-client-001-2102-20260723103001"
}
```

字段：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| `taskPlanId` | number | 是 | 要执行的任务计划 |
| `idempotencyKey` | string | 是 | 防止双击和重试重复创建 |

请求中不传地图坐标、机器人厂商、ROS2 接口、MQTT topic 或操作者信息。操作者从登录上下文取得。

响应：

```jsonc
{
  "taskExecutionId": "task-exec-20260723-0001",
  "taskPlanId": 2102,
  "taskCode": "RETURN_HOME",
  "displayName": "返回二楼驻点",
  "robotId": "robot-001",
  "status": "QUEUED",
  "createdAt": "2026-07-23T10:30:01+08:00"
}
```

## 4. 查询任务状态

```text
GET /api/bigscreen/panorama/task-executions/{taskExecutionId}
```

响应：

```jsonc
{
  "taskExecutionId": "task-exec-20260723-0001",
  "taskPlanId": 2102,
  "taskCode": "RETURN_HOME",
  "displayName": "返回二楼驻点",
  "robotId": "robot-001",
  "status": "RUNNING",
  "progress": 45,
  "currentStep": "NAVIGATE_TO_HOME",
  "currentStepName": "正在返回二楼驻点",
  "errorCode": null,
  "message": null,
  "startedAt": "2026-07-23T10:30:03+08:00",
  "completedAt": null,
  "updatedAt": "2026-07-23T10:30:20+08:00"
}
```

`progress` 和步骤信息在机器人端不支持时允许为空。

## 5. 取消任务

建议二期提供：

```text
POST /api/bigscreen/panorama/task-executions/{taskExecutionId}/cancel
```

请求：

```json
{
  "reason": "OPERATOR_CANCELLED"
}
```

是否允许取消以及不可取消步骤由管理端任务策略决定。

## 6. 机器人实时状态

机器人客户端通过现有统一状态链路上报任务和环境状态：

```jsonc
{
  "robotId": "robot-001",
  "controlMode": "NAVIGATION",
  "missionStatus": "RUNNING",
  "navigationStatus": "RETURNING_HOME",
  "dockingStatus": "UNDOCKED",
  "currentMapId": "map-floor-2",
  "currentMapVersion": "v12",
  "localizationStatus": "LOCALIZED",
  "activeTask": {
    "taskExecutionId": "task-exec-20260723-0001",
    "taskCode": "RETURN_HOME",
    "status": "RUNNING",
    "progress": 45,
    "currentStep": "NAVIGATE_TO_HOME",
    "errorCode": null,
    "message": null
  },
  "timestamp": "2026-07-23T10:30:20+08:00"
}
```

管理端消费机器人反馈并更新任务实例。大屏最终以管理端任务实例状态为准。

## 7. 状态枚举

| 状态 | 说明 |
|---|---|
| `QUEUED` | 等待下发 |
| `DISPATCHED` | 已下发 |
| `ACCEPTED` | 机器人已接受 |
| `RUNNING` | 正在执行 |
| `PAUSED` | 已暂停 |
| `SUCCEEDED` | 执行成功 |
| `FAILED` | 执行失败 |
| `CANCELLING` | 正在取消 |
| `CANCELLED` | 已取消 |
| `TIMED_OUT` | 执行超时 |

## 8. 错误码

| 错误码 | 说明 |
|---|---|
| `ROBOT_OFFLINE` | 机器人离线 |
| `TASK_NOT_BOUND` | 当前机器人未配置该任务 |
| `TASK_CONFLICT` | 正在执行冲突任务 |
| `MANUAL_CONTROL_ACTIVE` | 本体正在被人工控制 |
| `ESTOP_ACTIVE` | 急停已生效 |
| `CURRENT_MAP_UNKNOWN` | 当前地图未知 |
| `LOCALIZATION_UNAVAILABLE` | 定位不可用 |
| `HOME_POINT_NOT_CONFIGURED` | 当前地图未配置驻点 |
| `CROSS_MAP_NOT_SUPPORTED` | 不支持跨地图返航 |
| `ROBOT_NOT_DOCKED` | 机器人未停靠在充电桩 |
| `DOCK_UNAVAILABLE` | 充电桩不可用 |
| `UNSUPPORTED_STEP` | 机器人不支持任务步骤 |
| `PATH_PLANNING_FAILED` | 路径规划失败 |
| `DOCK_UNLOCK_FAILED` | 充电机构解锁失败 |
| `UNDOCK_PATH_BLOCKED` | 驶离区域被阻挡 |

## 9. 管理端依赖

现有查询接口：

```text
GET /api/v1/management/task-workflow-plans
GET /api/v1/management/task-workflow-instances
GET /api/v1/management/task-workflow-instances/{id}
```

仍需管理端确认：

- 按机器人查询快捷任务的接口。
- 人工立即执行任务计划的接口。
- 取消任务实例的接口。
- 管理端任务状态与机器人实时状态的同步方式。

管理端内部接口路径确定后，BFF 接口保持不变。
